> ##### 1. 프로젝트 설명
> ##### 2. 프로젝트 실행 화면 (Demo)
> ##### 3. Stopwatch UML Diagram

## 1. 프로젝트 설명
* 현재 시간을 기준으로 스탑워치가 시작.
 * 기능
    * **'q'** (STOP) : **q**입력시 실행중인 스탑워치가 종료되고 소요시간을 출력합니다.
    * **'h'** (HOLD) : **h**입력시 실행중인 스탑워치가 일시정지됩니다.
    * **'a'** (RERUN) : **a**입력시 **h**키를 눌러 일시정지했던 스탑워치가 이어서 실행됩니다.
